# CSS Flexbox 布局指南

## 什么是 Flexbox？

Flexbox 是一种一维布局模型，可以更有效地分配容器空间和对齐项目。

## 基础概念

### 启用 Flexbox

```css
.container {
  display: flex;
}
```

## flex-direction

```css
.container {
  display: flex;
  flex-direction: row;
}
```

## justify-content

```css
.container {
  display: flex;
  justify-content: flex-start;
}
```

## 总结

Flexbox 是现代 CSS 布局的重要工具。
